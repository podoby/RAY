# RAY
AI penetration testing assistant

to Run Ray :

```git clone https://github.com/podoby/RAY.git```

```cd RAY```

create and activate python environment:

```python3 -m venv venv```

```source venv/bin/activate```

make sure to upgrade your dependency:

```python3 -m pip install --upgrade pip```

```pip install -e .```

export your API Key:

```export MISTRAL_API_KEY="PUT IT HERE"```

*you can get one from here https://v2.auth.mistral.ai/login?flow=e4262488-63bf-4278-a4e6-b2114c51ab4e
